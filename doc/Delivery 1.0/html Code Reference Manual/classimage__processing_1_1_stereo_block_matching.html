void image_processing::StereoBlockMatching::process ( FrameList &  frames)
virtual

Performs stereo block matching using stereo block matching.

The stereo block matching algrothm used is the OpenCV semi-global block matcher (cv::StereoSGBM) This algortihm is based on one defined by Hirschmuller, H in 2008.

Hirschmuller, H. Stereo Processing by Semiglobal Matching and Mutual Information, PAMI(30), No. 2, February 2008, pp. 328-341
